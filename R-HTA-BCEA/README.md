# Introduction to R for Health Economics using BCEA

The slides for this lecture are available at [https://gianluca.statistica.it/slides/bcea](https://gianluca.statistica.it/slides/bcea). They are in `html` format --- it's just easier to access them (and make them accessible!) and, if you really would like that, you can create and download a pdf version.

From your browser, go to the [URL above](https://gianluca.statistica.it/slides/bcea) and click on the "hamburger" symbol in the left-bottom corner (that is the three horizontal lines <img src="bars-solid.svg" width="18">). The click on "Tools" on the left-hand side menu that opens and then on "pdf export mode. This creates a pdf version of the slides, which you can then print using your browser's facilities (e.g. CTRL+P).
